# Neil's Pet Cafe

A web-based reservation and management system for a pet-friendly cafe.

## Features

- User registration and login
- Reservation booking with calendar
- Menu browsing
- Announcements board
- Activity/order history
- Admin dashboard with analytics
- User management
- Approvals system
- Reports and data visualization
- System settings

## Technologies

- HTML5
- CSS3
- JavaScript (ES6+)
- Figma (UI/UX design)
- GitHub Pages (hosting)
